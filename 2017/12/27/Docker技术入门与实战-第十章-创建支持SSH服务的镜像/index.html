Docker技术入门与实战-第十章 创建支持SSH服务的镜像

2017-12-27
  • Docker
  • 进入容器可以在Host主机上使用attach、exec等命令
  • 通过SSH服务能解决远程管理容器的问题
  • 两种创建容器的方法：
    • docker commit
    • 基于Dockerfile创建

10.1 基于commit命令创建

  1. 安装openssh-server
  2. 修改root用户密码

  3. 辑配置文件/etc/ssh/sshd_config，允许以 root 用户通过 ssh 登录：

  4. 启动SSH servcie

  5. exit，退出容器

  6. 保存镜像:sudo docker commit newImageName repoName:Tag

    • sudo docker commit sshunbuntu sshd:ubuntu

  7. 使用镜像：sudo docker run -p 10022:22 -d sshd:ubuntu /otherScript.sh

  8. 在其他主机登陆docker：sudo ssh root@127.0.0.1 -p 10022

10.2 使用Dockerfile创建

  1. 编写Dockerfile

  2. 创建镜像

    sudo docker build -t sshd:ubuntu

  3. 运行容器

    sudo docker run -d -p 10022:22 sshd:ubuntu
